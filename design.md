# DESIGN BRIEF: The Pot Chronicles (Хроники Кастрюли)

## Visual Direction (reference-based)

Использовать визуальный стиль, вдохновлённый вывеской: reference.png

- крупный рубленный шрифт (геометрический, гротеск)
- строгая композиция и ровная верстка
- высокая контрастность (черный фон, белый текст)
- акцентный цвет (неоновые оттенки розового/фиолетового)
- ощущение современной городской минималистичной гастрономии
- легкое индустриальное настроение (матовый металл, бетон, ровные линии)

Ссылка на референс: вывеска "Чебуреки по рецептам ИИ" —
контрастный белый шрифт + яркий розово-фиолетовый элемент.

## Typography
Codex должен подобрать бесплатный webfont:
- Основной: строгий геометрический гротеск (например: Inter, Montserrat, Urbanist)
- Заголовки: крупный, жирный
- Body text: нейтральный, высокая читаемость

Все шрифты подключать локально или через Google Fonts.

## Color System
Codex должен использовать следующую палитру:

- **Основной фон:** почти черный (#0d0d0d — #111111)
- **Текст основной:** белый (#ffffff)
- **Вторичный текст:** серый (#cccccc)
- **Акцентный цвет:** яркий неон:
  - #ff00cc (фиолетово-розовый)
  - или #cc33ff (неон-фиолетовый)

Акцентный цвет использовать для:
- логотипа (если в тексте)
- ссылок при наведении
- кнопок
- разделителей
- небольших декоративных элементов

## Layout
Codex должен создать чистую минималистичную сетку:
- максимальная ширина контента ~860–900px
- большие поля (padding 48–80px)
- секции визуально разделены "воздухом"
- мобильная адаптация обязательна

Главная страница:
- крупный логотип/титул “Хроники Кастрюли / The Pot Chronicles”
- короткий подзаголовок
- блок категорий (генерируемых Python-скриптом)
- лента последних рецептов

## Logo / Wordmark
Codex должен создать текстовый логотип:
- "Хроники Кастрюли" крупно белым
- “Кастрюли” содержит акцент на одной или двух буквах (в референсе — цветной "ИИ")
- Допустимый вариант: подсветить буквы "ПО" или "РИ" акцентным цветом
- Латинская версия: "The Pot Chronicles" с аналогичной подсветкой

Логотип — чистый текст в CSS, без SVG, если не требуется.

## Hugo Implementation Tasks

Codex должен создать или обновить следующие файлы:

### 1. site/assets/css/style.css
- Вся основная типографика
- Цветовая схема
- Кнопки
- Ссылки
- Сетки и типографические отступы
- Компоненты меню
- Дизайн карточек рецептов
- Адаптивные стили

### 2. site/layouts/_default/baseof.html
- Подключение стилей
- Опорная HTML-структура (<main>, <header>, <footer>)

### 3. site/layouts/partials/header.html
- Логотип
- Главное меню
- Мобильная версия

### 4. site/layouts/_default/list.html
- Список рецептов
- Список категорий

### 5. site/layouts/_default/single.html
- Страница рецепта
- Большой заголовок
- Хорошая типографика шагов и ингредиентов

Codex должен обеспечить:
- аккуратный HTML
- семантику
- адаптивность
- отсутствие лишнего JS
- простоту расширения

## Non-Goals

- менять Python-скрипты
- менять структуру рецептов
- менять config.yaml
- добавлять сложные JS-фреймворки

- header.html
